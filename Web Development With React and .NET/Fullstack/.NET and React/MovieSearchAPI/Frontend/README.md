# Frontend

This folder contains the React + Vite client for **MovieSearchAPI**.

For full project setup, Docker usage, environment variables, and API documentation, see the root-level [`README.md`](../README.md).

## Available Commands

```bash
npm install
npm run dev
npm run build
npm run lint
```

## Local Development URL

- http://localhost:5173

During local development, Vite proxies `/api` requests to the backend at `http://localhost:5202`.
