# PURGE

PURGE is a single-player first-person action game developed in Unity.

## Overview
The player progresses through a series of combat-focused scenes, completes objectives, fights enemy units, and reaches a final boss encounter in the Reactor Cavern.

## Repository Scope
This repository is a lightweight portfolio/source snapshot of the project.
A full academic project archive and build package can be shared separately via cloud storage.

## Included
- Core gameplay scripts
- Scene flow
- Key prefabs
- Unity package configuration
- Project settings

## Build
- Engine: Unity 6
- Platform: Windows (64-bit)

## System Requirements
### Minimum
- OS: Windows 10 / 11 (64-bit)
- Processor: Intel Core i3 (4th Gen) or equivalent AMD CPU
- Memory: 8 GB RAM
- Graphics: DirectX 11 compatible GPU
- Storage: 3?5 GB available space
- Resolution: 1280x720

### Recommended
- OS: Windows 10 / 11 (64-bit)
- Processor: Intel Core i5 / AMD Ryzen 5 or better
- Memory: 8?16 GB RAM
- Graphics: NVIDIA GTX 1050 / AMD RX 560 or better
- Storage: SSD recommended
- Resolution: 1920x1080

> These system requirements are estimated based on the current build and have not been benchmarked across multiple target machines.
